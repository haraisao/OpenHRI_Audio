PulseAudioOutput
================
PulseAudioライブラリを使用した音声出力コンポーネント

:Vendor: AIST
:Version: 1.08
:Category: communication

Usage
-----

  ::

  $ pulseaudiooutput

   "AudioDataOut", "DataOutPort", "TimedOctetSeq", "音声データ出力"
